Letterboxd Localized
FR
Fetches localized titles & overviews from TMDB and injects them on Letterboxd.

No API key? See the API Key Help tab.

If you enjoy this extension, you can support its development on Ko-fi — it helps a lot!

Cache

Titles: 30d · Streaming & trailers: 7d · Cast: 30d

This extension needs a free API key from The Movie Database (TMDB) to fetch localized data.

1

Create a free account on TMDB

→ Sign up on TMDB
2

Go to your account Settings → API

→ Open API settings
3

Request a Developer key — fill the form with any URL (e.g. localhost).

4

Copy the API Key (v3 auth) and paste it in the Settings tab.
